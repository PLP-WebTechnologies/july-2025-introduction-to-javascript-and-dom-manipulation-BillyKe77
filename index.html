Web Development with JavaScript

Variables and Data Types

JavaScript has variables (let, const) and different data types.

Variable declaration
let message = "Hello, World!";
const pi = 3.14159;
let count = 0;

Data types
let stringType = "Text";
let numberType = 42;
let booleanType = true;
let arrayType = [1, 2, 3];
let objectType = {name: "John", age: 30};
